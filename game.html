Play Piano
  1. To play a game click the 'Start' button.

    Two other buttons will appear in place of "Start" button: the "Restart" and "Level 1" buttons.
    The "Restart" button allows to restart the game. The 'Level' button indicates beginning of the level one.
  2. The randomly picked sequence of keys will flash and play sound at the same time. The user will have to repeat the patten of shown keys.
    If the keys are correct, the level will increase by one and an additional key will be added to sequence.

    But if the user clicks the wrong key in a sequence, the user will hear a beeping sound and the number in the level box will change to a triple exclamation mark.

    The series of keys will be played again until user successfully repeat the pattern of keys and level up.
  3. When the user reaches Level 20, the message 'YOU WON!!!' will appear in a level box and a cheering sound will be played.

    After that the game will reload.
